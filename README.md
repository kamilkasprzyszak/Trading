# Finance, Algorithmic Trading & Risk Frameworks

Repozytorium stanowi skonsolidowane portfolio moich projektów, skryptów oraz narzędzi analitycznych poświęconych inżynierii finansowej, handlowi ilościowemu (Quantitative Trading) oraz systemowemu zarządzaniu kapitałem. 

Głównym celem projektu jest połączenie matematyki finansowej, analizy danych rynkowych oraz programowania w celu poszukiwania powtarzalnych przewag statystycznych i automatyzacji procesów decyzyjnych.


## 💻 Stack Technologiczny

* **Języki programowania:** Python, Pine Script (v5/v6), MQL5, Bash
* **Analiza danych & Statystyka:** Pandas, NumPy, Statsmodels, SciPy
* **Komunikacja & Narzędzia:** REST API, WebSockets, Git

---

## 📈 Metodologia Pracy

Wszystkie zawarte tutaj projekty łączy wspólny mianownik: **podejście oparte na danych (Data-Driven)**. Każda koncepcja rynkowa przechodzi przez rygorystyczny proces walidacji statystycznej. Nadrzędną zasadą jest ochrona kapitału, dlatego warstwa zarządzania ryzykiem jest technicznie niezależna i nadrzędna nad modułami egzekucyjnymi.
